# flutter_calendar

最近我们的UI小姐姐给了一份这样的日历设计图 ┭┮﹏┭┮，  
可以上下滑动，支持多日选择，再次进入日历页面可以选中上次选中的日期，  
开始想冒着侥幸的心里去找找网上的开源库，  
无奈找了许久找不到可以上下滑动的日历，  
故花了三天时间终于写完了初版
